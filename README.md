# Bangla-Letter-Recognition-CNN
CNN-based Bangla handwritten letter recognition (36 classes) using 64×64 grayscale images. Model includes 3 Conv2D + MaxPool layers followed by dense classifier. Achieved 91% test accuracy with macro F1 ≈0.90. Detailed per-class precision/recall reported, highlighting strong performance across most vowels and consonants.
